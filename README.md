# Employee Management Portal

## Project Overview

Employee Management Portal is a web application developed during the TECHSHIP Internship Program using Next.js and React. The project was created to learn modern web development concepts and build a basic employee management system interface.

Currently, the application contains Sign Up, Login, and Add Employee pages. MongoDB integration was explored but is not fully completed.

## Features

- User Sign Up Page
- User Login Page
- Add Employee Form
- Responsive UI Design
- Next.js Routing

## Technologies Used

- HTML
- CSS
- JavaScript
- React.js
- Next.js
- Tailwind CSS
- MongoDB (Learning & Setup Attempt)

## Installation Instructions

### Clone Repository

git clone https://github.com/yourusername/employee-management-portal.git

### Install Dependencies

npm install

### Run Development Server

npm run dev

### Open Browser

http://localhost:3000
